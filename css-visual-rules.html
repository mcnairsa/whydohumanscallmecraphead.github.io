1 of 10 Introduction

Yeah, there's a bucket load of stuff ... Note that later lessons go into all this in more detail. This lesson is pretty much just an overview.

2 of 10 Font Family

A font-family is a group of related fonts. You set the font family for the <p> elements like this: 

    p {
	  font-family: Georgia;
      }
There's a bunch of font stuff on Dan's Tools. Enjoy!

3 of 10 Font Size

One way to set the size of the text is by pixels (originally pixel was shorthand for 'picture cell'). 

    p {
	  font-size: 18px;
      }
Will set the text size to 18 pixels. Note that pixel is a absolute size. There are better ways to play with the size of your text.

4 of 10 Font Weight

Weight means how light or dark the text is. Like making it bold but with much more fine-grained control. E.g.: 

  * {
	font-weight: bold;
	}
... will make everything bold. To set certain parts back you can use the value of normal instead of bold. Again see later lessons for how to really have fun with font weights.

5 of 10 Text Align

Text can be aligned to the left, centre, right and all manner of lovely positions. (As usual see MDN for details). 

p {
  text-align: center;
  }
Will center the text of all <p> elements on the page.

6 of 10 Color and Background Color

The colour of foreground (i.e. text++) and background are set independently. 

.caption {
    background-color: white;
    color: black;
  }
Note: it's not a good idea to set the background of your text editor to the same as the text colour.

7 of 10 Opacity

Opacity is a measure of how visible something is. If it is fully opaque it's solid. The opposite is transparent. Values for the opacity property range from 0 (transpaent or invisible) to 1 for 100% opacity. A value in between will allow an over-lapped element, an image for example, to be partly visible. 

.caption {
	background-color: white;
	color: black;
	opacity: 0.75;
  }
... gives a much pleasanter viewing experience, don't you think?

8 of 10 Background Image

We can use an image as a background thus: 

.main-banner {
    background-image: url('images/mountains.jpg');
  }
The path can be absolute if it is an external file or relative if the file is local to the html file.

9 of 10 Important

A declaration, i.e. one part of a ruleset, can be marked as !important which will tend to override any other rules or declarations. Remember though, that there are always several style sheets in operation. Generally, the ... is regarded as a Very Bad Thing. It can usefully be used though in a user style sheet, perhaps for accessibility. See the MDN !important page for more info.
