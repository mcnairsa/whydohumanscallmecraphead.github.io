1 of 5 Introduction

The standard box model is additive. The padding, border and margins increase the size of the element's box. This can cause problems if the page starts to get busy. Also, because height and width don't mean what they say or appear to mean, we have to take all the values (not just the values for height and width) into account when working out how much elements will add up to on the page.
